# 功能演示指南

## 启动应用

```bash
npm run dev
```

访问 http://localhost:5173

## 功能演示流程

### 1. 语言切换

**位置**：页面右上角地球图标

**操作步骤**：
1. 点击右上角的地球图标 🌐
2. 从下拉菜单中选择语言
   - English（英语）
   - 中文（简体中文）
3. 页面会立即切换到选择的语言
4. 语言偏好会自动保存

**验证**：
- ✅ 所有 UI 文本都应该切换到对应语言
- ✅ 提示信息使用对应语言
- ✅ FAQ 和特性展示使用对应语言
- ✅ 刷新页面后语言保持不变

### 2. 图片上传

**方式一：拖拽上传**
1. 准备一些测试图片（PNG、JPG、GIF、BMP、WebP）
2. 将图片拖拽到上传区域
3. 释放鼠标，图片会自动添加到列表

**方式二：点击选择**
1. 点击上传区域
2. 在文件选择对话框中选择图片
3. 支持多选（按住 Ctrl/Cmd 键）
4. 点击"打开"

**限制**：
- 单次最多 20 张图片
- 支持的格式：PNG、JPG、JPEG、GIF、BMP、WebP

**验证**：
- ✅ 图片成功添加到列表
- ✅ 显示原图预览
- ✅ 显示文件名和大小
- ✅ 状态显示为"待处理"

### 3. 配置压缩参数

**位置**：左侧设置面板

**目标格式**：
1. 点击"目标格式"下拉菜单
2. 选择转换后的格式
   - 保持原格式
   - JPG
   - PNG
   - WebP
3. 如果选择"保持原格式"，只压缩不转换

**压缩质量**：
1. 使用滑块调节质量（1%-100%）
2. 或直接在输入框中输入数值
3. 默认值：92%
4. 质量越高，文件越大，画质越好

**验证**：
- ✅ 滑块和输入框同步
- ✅ 输入框只接受 1-100 的数值
- ✅ 设置会应用到所有图片

### 4. 处理图片

**增量处理**（推荐）：
1. 点击"开始处理"按钮
2. 只处理状态为"待处理"的图片
3. 已处理的图片不会重复处理
4. 适合分批上传和处理

**重新生成**：
1. 修改压缩参数后
2. 点击"重新生成"按钮
3. 按新参数重新处理所有图片
4. 适合调整参数后重新处理

**处理过程**：
- 状态显示为"处理中..."
- 显示处理进度
- 处理完成后显示"已完成"

**验证**：
- ✅ 显示处理后的预览图
- ✅ 显示压缩后的文件大小
- ✅ 显示压缩率（例如：-45%）
- ✅ 可以对比原图和压缩后的图片

### 5. 下载图片

**单个下载**：
1. 在图片卡片上点击"下载"按钮
2. 浏览器会自动下载该图片
3. 文件名格式：`compressed_原文件名.扩展名`

**批量下载**：
1. 点击左侧面板的"下载全部"按钮
2. 只下载已完成处理的图片
3. 每个图片会单独下载

**验证**：
- ✅ 下载的图片可以正常打开
- ✅ 文件大小确实减小了
- ✅ 图片质量符合预期
- ✅ 格式转换正确

### 6. 删除图片

**单个删除**：
1. 在图片卡片上点击"删除"按钮
2. 图片会从列表中移除

**清空列表**：
1. 点击左侧面板的"清空列表"按钮
2. 所有图片会被移除

**验证**：
- ✅ 图片从列表中消失
- ✅ 统计信息更新
- ✅ 内存被释放

## 性能测试

### 1. 压缩效果测试

**测试图片**：
- PNG 图片（通常压缩率最高）
- JPG 图片（已压缩格式）
- 大尺寸图片（>5MB）
- 小尺寸图片（<100KB）

**预期结果**：
- PNG → JPG：压缩率 50-80%
- PNG → WebP：压缩率 60-90%
- JPG → JPG：压缩率 10-30%
- JPG → WebP：压缩率 20-40%

### 2. 处理速度测试

**测试场景**：
- 单张小图片（<1MB）：<1 秒
- 单张大图片（5-10MB）：2-5 秒
- 10 张中等图片（2-3MB）：10-20 秒
- 20 张小图片（<1MB）：5-10 秒

**注意**：
- 处理速度取决于设备性能
- 浏览器性能影响处理速度
- 图片尺寸和格式影响处理时间

### 3. 内存使用测试

**测试步骤**：
1. 打开浏览器任务管理器
2. 上传 20 张大图片
3. 处理所有图片
4. 观察内存使用情况
5. 清空列表
6. 验证内存释放

**预期结果**：
- 处理过程中内存增加
- 清空列表后内存释放
- 没有明显的内存泄漏

## 常见问题排查

### 问题：图片上传失败

**可能原因**：
- 文件格式不支持
- 文件损坏
- 浏览器权限问题

**解决方案**：
- 检查文件格式
- 尝试其他图片
- 检查浏览器控制台错误

### 问题：处理失败

**可能原因**：
- 图片过大
- 浏览器内存不足
- 图片格式特殊

**解决方案**：
- 减少图片数量
- 关闭其他标签页
- 尝试其他浏览器

### 问题：下载失败

**可能原因**：
- 浏览器下载权限
- 磁盘空间不足
- 浏览器设置问题

**解决方案**：
- 检查浏览器下载设置
- 清理磁盘空间
- 尝试单个下载
